== Description ==

Elevate your Website with the Dynamic Keyword Insertion for WP Plugin by BVNode. This innovative tool allows you to customize web page text based on the search terms used by visitors, improving your ads Quality Score and ensuring a personalized, engaging user experience.
* Lower Your CPC Cost,
* Boost Your Ad Rank
* Influence Your Ad Performance

== Basic Usage ==

`[dki4wp param="state" default="The best pizza in California! Buy now!" 
prefix="The best pizza in " suffix="! Buy now!"]`

When the query parameter **state** is empty and the page or post does not use any **set** with the parameter **state** results in:

*The best pizza in California! Buy now!* 

and if **state** is **Texas** results in:

*The best pizza in Texas! Buy now!*

== Installation ==

= INSTALL DKI4WP FROM WITHIN WORDPRESS =

1. Visit the plugins page within your dashboard and select ‘Add New’;
1. Search for ‘Dynamic Keyword Insertion for WP’;
1. Activate Dynamic Keyword Insertion for WP from your Plugins page;

= INSTALL DKI4WP MANUALLY =

1. Upload the dynamic-keyword-insertion-for-wp’ folder to the /wp-content/plugins/ directory;
1. Activate the Dynamic Keyword Insertion for WP plugin through the ‘Plugins’ menu in WordPress;
